# rasporyadok_dnya
Проект "Распорядок дня" — это простая и стильная веб-страница, посвящённая практическим советам по организации ежедневного расписания для повышения продуктивности и благополучия. Страница включает введение, разделы по времени суток (утро, день, вечер) с конкретными рекомендациями, дополнительные советы и footer. Дизайн выполнен в фиолетовой гамме с градиентным фоном, разнообразными оттенками для секций, тенями и декоративными цветами по бокам для визуального комфорта. Идеально подходит для мотивации и самоорганизации.
